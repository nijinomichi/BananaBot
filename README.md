## Quick Start

1. Clone this repo
2. Copy `.env.example` → `.env` and fill in your secrets
3. Install dependencies: `pip install -r requirements.txt`
4. Run on Replit (recommended) or locally

> ⚠️ Never commit your real `.env` file. It's in `.gitignore`.
